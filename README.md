# 🛒 FinalProject – E-Commerce Web Application

This repository contains my **Final Project from Matrix Academy**.
The project represents a **functional prototype of an online store**, built to simulate a real-world e-commerce platform.

The application includes **user authentication, product management, an admin panel, and a responsive interface**, demonstrating both front-end and back-end development skills.

---

# 🚀 Project Overview

The goal of this project was to design and implement a **full web application similar to a real online shop**.

Key features include:

* User registration and authentication
* Secure login system
* Product catalog
* Admin dashboard for managing products
* Responsive design for different devices
* Basic e-commerce workflow

This project helped me understand how **modern web applications are structured and managed**.

---
